SuppressionListReader

public class SuppressionListReader<T : EmailEventRepresentable & Decodable>: Request<[T]>

The SuppressionListReader class is base class inherited by requests that retrieve entries from a supression list. You should not use this class directly.

    Parameters

    start

    Limits the search to a specific start time for the event.
    end

    Limits the search to a specific end time for the event.
    range

    A range of dates to search between If nil, the entire bounce list will be searched.
    page

    A PaginationInfo instance to limit the search to a specific page. The limit value cannot exceed 500. If not specified, the limit will be set to 500 and the offset will be set to 0.
